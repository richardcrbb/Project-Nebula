Welcome to Project Nebula

This project turns a Raspberry Pi 5 into a fully functional NAS and self-hosted server, running Ubuntu Server on a Btrfs filesystem for advanced features like snapshots, compression, and data integrity. It uses Docker to manage and isolate multiple services, including:

  • Nextcloud for private file storage and synchronization
  • WireGuard VPN for secure remote access
  • Pi-hole for network-wide ad blocking
  • Additional tools and services for automation, monitoring, and backups

Designed for reliability, low power consumption, and easy recovery, this setup provides a compact yet powerful home server solution.
